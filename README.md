# MONSTRO DAO — Smart Contracts

This repository contains the canonical smart contracts for the **MONSTRO DAO** protocol.

The contracts defined here govern the core on-chain mechanics of the MONSTRO ecosystem, including the protocol token, staking systems, and related infrastructure. This repository serves as the authoritative reference for contracts governed by MONSTRO DAO.

## Repository Structure

- `token/` — Protocol token contracts  
- `staking/` — Staking and reward distribution contracts  

Additional modules, interfaces, and utilities may be added as the protocol evolves.

## Status

These contracts are under active development and are subject to change.

Unless explicitly stated otherwise, the contracts in this repository:
- are not finalized
- have not been audited
- should not be considered production-ready

Final deployment addresses and audit reports (if applicable) will be communicated through official MONSTRO DAO channels.
